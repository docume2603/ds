# Carter Buschardt : 野外監視カメラが捉えた Bigfoot の手の映像 ⇒ この謎を解く（＋追加）


## 要旨

Bigfoot が出没すると期待されている場所に野外監視カメラが設置してある。その電池とメモリーカードを定期的に交換する作業を行っていたが、交換を終えてその場を離れようとした時に Bigfoot の手が映り込んだという。その映像（下）が彼の本の表紙にもなっている。

最初の映像の時刻と二枚目の映像の時刻は 4秒しか隔たっていない。この映像が撮影された当時、Bigfoot が付近に存在したという具体的な兆候（姿、物音、臭いなど）は皆無だった。

文字起こし＋日本語訳をつけておく。ついでに、この「Bigfoot の手」の映像の謎を解く。

## この謎を解く

Carter Buschardt はこの映像を彼の本の表紙に採用している。彼はこのネタをこの Dave Scott だけではなく、他の幾つものインタビュー、たとえば下

<a href="/ds/img/20240410_cb_bookcover.jpg" target="_blank"><img alt="20240410_cb_bookcover.jpg" border="0" src="/ds/img/20240410_cb_bookcover.jpg" width="90%"/></a>

でも取り上げている。誰もがこの映像を Bigfoot の手だとしか見えなくなってしまっている。

だが、余計な先入観を排除しさえすれば、この映像を一瞥しただけであっさり謎が解ける。簡単すぎるので 80%ルールを適用するまでもない。

正体は下のような防寒用（季節が冬であることに留意）作業上着。この上着を束ね、（監視カメラを設置した）木の枝に引っ掛けて、交換作業を行った。交換作業を終えて、その上着を枝から引き下ろした時に、監視カメラの視野をその上着の下部（裾）が横切った。さらに監視カメラに上着がぶつかった。だから、二枚目の映像の上部が振動で少し歪んでいる。

この映像をチェックしたのは、次回の交換作業の後であるから数か月後である筈。当然、上着を枝に引っ掛けた事など覚えていないだろうし、覚えていたとしてもそれに思い至らないのだろう。
